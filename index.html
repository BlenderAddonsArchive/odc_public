Blender Dental

Addon for Blender meant to streamline dental visualization, analysis and CAD and design workflows

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Blender Dental.

This is the webpage and code repository for Open Dental CAD and related scripts for dental applications of Blender. This is an addon and script collection which evolved out of a 3d addiction hobby during dental school.

Getting Started

Download the source .zip package, and head over to the Wiki for installation instructions.

Community Support

We are building a community forum over at ZoHo where users can share cases, ask each other for help and exchange ideas. https://www.zohodiscussions.com/blenderdental#home

Contributors

@patmo141 is the main author and developer for Open Dental Cad, however I would love any contributions, pull requests and bug reports! The best way to help is to contribute documentation to the wiki and detailed bug reports.
